An Interactive Explorer for Minibar Beer Data

Interact with the widgets on the left to query a subset of beers to plot. The color widget will be a scale for numeric color, and categorical for other data types. The Jitter widget defines the jitter for both x- and y-scales, and select a portion of the graph will filter the table as well. Selecting a blank area on the graph will reset the selection filter. Inspired by the Bokeh Movie Explorer.
